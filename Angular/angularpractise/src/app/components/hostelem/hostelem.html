5. Binding Collisions

Rule 1 — both static → parent wins

Ex. parent says role="group" and host says role="slider" → role="group" parent wins

Rule 2 — one static one dynamic → dynamic wins

Ex. parent says role="group" (static) and host says [attr.role]="myRole" (dynamic) → dynamic host wins

Rule 3 — both dynamic → host wins

Ex. parent says [id]="parentId" and host says [id]="hostId" → host wins

Check DevTools to see which value is applied on <app-hostelem> tag
