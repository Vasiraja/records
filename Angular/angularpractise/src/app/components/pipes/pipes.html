5. Pipes

Transform date in the template

..1.. Uppercase | Lowercase

{{"Vasiraja" | uppercase}}

{{"vasiRaja" | lowercase}}

..2.. Date Pipe

{{ today | date }}

{{ today | date:'shortDate' }}

{{ today | date:'fullDate' }}

..3.. Currency

{{amount | currency}}

{{amount | currency:'INR'}}

{{ amount | currency:'USD':'symbol':'1.4-5' }}

..4.. Value

{{ value | number:'1.2-3' }}

..5.. Percentage

{{score | percent}}

..6.. Slice

{{"VASI RAJA" | slice:0:6}}

..7.. JSON

{{user | json}}

..8.. Asyc
